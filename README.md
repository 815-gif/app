# Mis Manhwas Leídos

Aplicación web simple (HTML/CSS/JS) para guardar los manhwas que ya leíste usando `localStorage`.

## Funciones

- Guardar manhwa con: título, autor, artista, tipo, demografía, género, estado, capítulos leídos, total de capítulos y nota.
- Soporte de portada por URL o **subiendo imagen desde tu galería** (`input file`).
- Ver **detalles** de cada manhwa en una ventana modal.
- Añadir **valoración** (0 a 5) y verla como número + estrellas pintadas.
- **Editar** y **eliminar** cualquier manhwa guardado.
- Tema visual con toque **rosa de flores de cerezo + turquesa**.
- Barra de navegación con iconos de **Home**, **Biblioteca** y **Perfil**.
- Perfil con estadísticas automáticas: total guardados, capítulos leídos, promedio de valoración, estado frecuente, géneros top y último agregado.

## Cómo usar

1. Abre `index.html` en el navegador.
2. Completa el formulario y pulsa **Guardar**.
3. En cada tarjeta puedes usar: **Ver detalles**, **Editar** o **Eliminar**.

## Datos

Los datos se guardan en tu navegador con la clave: `manhwas-leidos`.
